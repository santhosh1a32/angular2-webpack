# angular2-webpack
Just download or clone the code and run the following commands

To install all the dependencies<br/> 
<em>npm install</em>

To build angular 2 project<br/> 
<em>npm run build</em>

For production build<br/> 
<em>npm run prod</em>

To run project locally using webpack dev server<br/> 
<em>npm start</em>
